# 列表滚动功能修复

## 🐛 问题描述

用户反馈：
- 列表很长，占用大量垂直空间
- 列表不可滚动，所有项目都显示出来

## ✅ 修复方案

### 1. 添加滚动容器

为列表添加可滚动容器，设置最大高度：

**修复前：**
```javascript
h('div', { className: "space-y-2" },
    // 添加按钮
    h('button', ...),
    // 列表项（直接渲染，无高度限制）
    exportExtras.map(item => ...)
)
```

**修复后：**
```javascript
h('div', { className: "space-y-2" },
    // 添加按钮（固定在顶部）
    h('button', ...),
    // 列表项（可滚动容器）
    h('div', { 
        className: "max-h-64 overflow-y-auto space-y-2 pr-1 scrollable-list"
    },
        exportExtras.map(item => ...)
    )
)
```

### 2. 样式设置

- ✅ `max-h-64` - 最大高度 16rem (256px)
- ✅ `overflow-y-auto` - 垂直方向自动滚动
- ✅ `space-y-2` - 列表项间距
- ✅ `pr-1` - 右侧内边距，为滚动条留出空间
- ✅ `scrollable-list` - 自定义滚动条样式类

### 3. 自定义滚动条样式

添加了 `scrollable-list` CSS 类，提供：
- ✅ 细滚动条（6px 宽度）
- ✅ 橙色主题滚动条
- ✅ 悬停效果
- ✅ 跨浏览器兼容（Chrome/Safari/Firefox）

## ⚙️ 自定义高度

如果需要调整最大高度，可以修改 `max-h-64`：

- `max-h-48` - 192px（更紧凑）
- `max-h-64` - 256px（当前设置）
- `max-h-80` - 320px（更宽松）
- `max-h-96` - 384px（非常宽松）
